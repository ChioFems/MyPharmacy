FOR NEW PROJECT:
1. Run cmd in server folder: npm init -y
This will initialize npm package

2. Install the following packages: npm i express body-parser cors dotenv helmet morgan mongoose nodemon
Express for APIs framework
Body-parser for parsing the data that is coming in
Cors for cross origin resource sharing
Dotenv for environment variables
Helmet fro protecting the APIs
Morgan for logging the API calls
Mongoose for handling mongoose db calls
Nodemon for live server relay
