FOR NEW PROJECT:
1. Run cmd in server folder: npx create-react-app appName
This will initialize npm packages for the frontent app

2. Install the following packages: npm i @mui/material @emotion/react @emotion/styled @mui/x-data-grid @mui/icons-material react-router-dom@6 react-pro-sidebar@0.7.1 formik yup @fullcalendar/core @fullcalendar/daygrid @fullcalendar/timegrid @fullcalendar/list @nivo/core @nivo/pie @nivo/line @niv0/bar @nivo/geo react-redux @reduxjs/toolkit react-datepicker  
  
@mui, @emotion for Material UI
NB: If datepicker version does not work, use npm i react-datepicker@4.8.0

3. Then run: npm audit fix --force
